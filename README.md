# 🚀 Maxura Agency — AI-Powered Lead Generation System

Maxura Agency is an AI-driven SMMA focused on generating high-quality leads for home remodeling and renovation businesses.

This repository contains the landing page and foundational system for building scalable client acquisition workflows using AI.

---

## 🧠 Project Overview

The goal of this project is to create a repeatable and scalable system that:

- Generates leads using paid advertising (Meta & Google Ads)
- Converts traffic using high-converting landing pages
- Automates follow-ups and workflows using AI tools
- Builds a predictable pipeline for service-based businesses

---

## ⚙️ Core Features

- Conversion-focused landing page
- Clear service offer for home remodeling niche
- AI-assisted copywriting and content generation
- Funnel structure for capturing and qualifying leads
